This C program simulates a Café Order Management System using 1D arrays, loops, and conditional statements.
The program displays a fixed menu of 5 items (Coffee, Tea, Sandwich, Burger, Pastry) with their prices.

It allows up to 10 customers to place orders. For each customer:

The user enters how many items they want.

For each item, the program takes the item number and quantity.

It then calculates and displays the total bill for that customer.

The program keeps track of:

Total quantities sold for each item,

Total revenue for the café, and

Determines which item was most ordered and least ordered.

Finally, it displays a summary showing:

Total revenue,

Total items sold,

Most ordered item,

Least ordered item.
