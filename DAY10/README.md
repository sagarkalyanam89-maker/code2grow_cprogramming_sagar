The program calculates bills for multiple customers in a canteen.

It first reads the total number of customers.

For each customer:

It asks for the number of items purchased.

For each item, it requests an item code and quantity.

Based on the code entered, it assigns a fixed price:

Code 1 → Sandwich (80)

Code 2 → Burger (120)

Code 3 → Pizza slice (150)

Code 4 → Coffee (60)

Code 5 → Juice (50)

It computes the total cost for that customer by summing all item prices.

If the total exceeds 500, a 10% discount is applied.

It displays the bill for the customer.

After processing all customers, it prints a summary showing:

Total customers served.

Total revenue collected from all customers.
